InclusiFund Templates Environment

The web environment has been initialized. You can upload your HTML files to the public/ directory to serve them statically.

How to use:

  1. Upload your HTML files (like dashboard.html, login.html) to the public/ folder.
  2. Upload your CSS to public/assets/theme.css.
  3. Upload your JS to public/assets/js/.
  4. Access them directly via the URL (e.g., /dashboard.html).

Ready to go!

The Vite development server is running and will serve any files placed in the public/ directory automatically.
